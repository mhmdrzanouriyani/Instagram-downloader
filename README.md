## ✨ Features

- ✅ Download photos and videos from public profiles
- ✅ Works on Windows, Linux, macOS
- ✅ Simple CLI interface
- ✅ Saves to organized folders by username

---

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/mhmdrzanouriyani/Instagram-downloader.git
cd Instagram-downloader
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run it
```bash
python instadownloader.py
```

## ⚙️ Requirements

- Python 3.7+
